Clicker Adventure 2: The First One

About

As a collaborative academic project, our team developed "Clicker Adventure 2," an idle clicker game. The game features a core progression system where players combat monsters to earn gold and experience points. These resources fuel character development, allowing players to level up, unlock new abilities, and advance through distinct areas, each culminating in a final boss battle.

My Role

  • Engineered the core combat system, implementing mechanics for player attacks, damage calculation, and enemy interactions.
  • Developed the character management system, which included a creation module for player customization and a save/load feature for data persistence.
  • Designed a dynamic enemy system featuring randomized monster generation and a scaling difficulty curve to ensure balanced gameplay.
